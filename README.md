# Neumorphism Circular Loader

Projeto desenvolvido com HTML e CSS que cria uma animação de carregamento circular inspirada no conceito de Neumorphism. O componente utiliza sombras suaves e animações contínuas para produzir um efeito visual moderno e elegante.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- CSS Animations

## ✨ Funcionalidades

- Loader circular animado
- Rotação contínua
- Efeito Neumorphism
- Design minimalista
- Animação suave
- Sem utilização de JavaScript

## 📂 Estrutura do Projeto

```bash
Neumorphism-Circular-Loader/
│
├── index.html
├── css/
│   └── style.css
└── README.md
```

## 🎯 Como Executar

1. Clone ou baixe o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` no navegador.
